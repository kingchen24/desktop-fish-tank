# 🐟 桌面小鱼缸

一款轻量化桌面挂机养鱼小游戏，灵感来自《TBH》《Banana》《Rusty's Retirement》等桌面放置类游戏。

## ✨ 特性

- 🐠 **8种鱼类**：从小丑鱼到小鲸鱼，越贵的鱼产币越快
- 🫧 **点击泡泡**：鱼会吐泡泡，点击收集泡泡币
- ⏱️ **完全挂机**：鱼自动游泳、自动产币，关掉也没关系
- 💰 **离线收益**：再次打开时，小鱼们已经帮你赚了很多（最多累计8小时）
- 💾 **自动存档**：每30秒自动保存，关闭页面也会存
- 🎨 **精美渲染**：水纹、光线、海草、石子、粒子特效
- 🏪 **鱼店系统**：攒够泡泡币购买新品种

## 🚀 快速开始

### 方式一：浏览器直接运行

双击 `index.html` 或在浏览器打开即可！

### 方式二：Electron 桌面应用

```bash
# 安装依赖
npm install

# 开发运行
npm run dev

# 打包 Windows 便携版
npm run dist:win
```

## 🎮 操作说明

| 操作 | 说明 |
|------|------|
| 点击泡泡 | 收集泡泡币 |
| 按 S 键 | 打开/关闭鱼店 |
| 右键画布 | 重置游戏数据 |

## 🏗 技术栈

- 纯前端：HTML + CSS + JavaScript（Canvas 2D）
- 桌面包装：Electron
- 存储：localStorage
